# Quotebot ERP Frontend

React + TypeScript frontend for the Quotebot ERP system.

## Quick Start

```bash
npm install
npm start
```

## Pages

- Dashboard (/dashboard)
- User Permissions (/user-permissions)
- Quotations (/quotations)
- Products (/products)
- Analytics (/analytics)
- RFQ Inbox (/rfq-inbox)
- Client Ledger (/client-ledger)
- System Config (/system-config)

## Tech Stack

- React 18 + TypeScript
- React Router
- Tailwind CSS
- Material Symbols Icons
